RGLoader Realtime Patcher 
by tydye81

Features
-----------------
Supports dynamically enabling patches on xam or any other exe/dll in memory. 

Block_Live_Dns = Blocks xbox live DNS so you can't "phone home"
Map_USB_Mass = Makes all usb devices show up in xbox neighborhood
Default_Dashboard = Sets the default dashboard

Usage
-----------------
Place XtweakXam.xex and rgloader.ini onto the root of your hard drive. (Partition 1).
Reboot.
